# Game Backend Demo

> [!WARNING]
> Work In Progress

Multiplayer backend built with ASP.NET for a Unity game.

Features:
- ✅ Authentication (guest login)
- ✅ Matchmaking queue (2-player)
- ✅ Server-authoritative game sessions (tic-tac-toe)
- ✅ Match state API (board, turns, win/draw detection)
- 🔲 Admin dashboard (Bubble) — in progress
- 🔲 Persistence (database)
- 🔲 Disconnect handling
